# Лабораторная работа №6 — LLM-ассистент в консоли

## Цель работы
- Изучить принципы работы больших языковых моделей (LLM)
- Реализовать консольное приложение для диалога с LLM через локальный API (Ollama)
- Научиться управлять историей сообщений (контекстом диалога)
- Продемонстрировать генерацию текста на русском/английском языке

## Используемые технологии
- Python 3.8+
- `requests` — для HTTP-запросов к Ollama
- `ollama` — сервер для запуска LLM локально (LLaMA, Mistral, Gemma и др.)

## Структура проекта

```
llm_assistant/
├── main.py          # точка входа, CLI чат
├── llm_client.py    # класс для работы с Ollama API
├── requirements.txt # зависимости
└── README.md        # описание (этот текст)
```

## Принцип работы

1. Пользователь запускает приложение.
2. Приложение подключается к локальному серверу Ollama (по умолчанию `http://localhost:11434`).
3. Пользователь вводит сообщение.
4. Приложение отправляет запрос к выбранной модели вместе с историей диалога.
5. Модель генерирует ответ, который выводится на экран.
6. Диалог продолжается до команды выхода (`/exit`).

## Установка и запуск

### 1. Установите Ollama

Скачайте и установите Ollama с официального сайта:


**llm_client.py** – класс для общения с Ollama

```python
import requests
import json

class OllamaClient:
    def __init__(self, base_url="http://localhost:11434", model="llama3.2:3b"):
        self.base_url = base_url
        self.model = model
        self.history = []  # хранит список сообщений: {"role": "user"/"assistant", "content": "..."}

    def add_message(self, role, content):
        """Добавить сообщение в историю"""
        self.history.append({"role": role, "content": content})

    def clear_history(self):
        """Очистить историю диалога"""
        self.history = []

    def generate(self, user_input, system_prompt=None):
        """
        Отправить запрос модели с учётом истории.
        Возвращает ответ модели (строка).
        """
        # Добавляем пользовательский ввод в историю
        self.add_message("user", user_input)

        # Формируем payload для Ollama API
        payload = {
            "model": self.model,
            "messages": self.history.copy(),
            "stream": False,
            "options": {
                "temperature": 0.7,
                "num_predict": 512
            }
        }
        if system_prompt:
            payload["system"] = system_prompt

        try:
            response = requests.post(f"{self.base_url}/api/chat", json=payload)
            response.raise_for_status()
            data = response.json()
            assistant_reply = data["message"]["content"]
            # Добавляем ответ ассистента в историю
            self.add_message("assistant", assistant_reply)
            return assistant_reply
        except requests.exceptions.ConnectionError:
            return "Ошибка: не удалось подключиться к Ollama. Убедитесь, что сервер запущен (ollama serve)."
        except Exception as e:
            return f"Ошибка при генерации: {str(e)}"
```

**main.py** – консольный интерфейс

```python
from llm_client import OllamaClient

def main():
    print("Добро пожаловать в LLM-ассистента!")
    print("Используется локальная модель через Ollama.")
    print("Команды:")
    print("  /exit   - завершить работу")
    print("  /clear  - очистить историю диалога")
    print("  /model <имя> - сменить модель (например /model llama3.2:3b)")
    print("-" * 50)

    client = OllamaClient()
    system_prompt = "Ты полезный AI-ассистент. Отвечай кратко, ясно и по-русски."

    while True:
        try:
            user_input = input("\nВы: ").strip()
            if not user_input:
                continue

            if user_input == "/exit":
                print("До свидания!")
                break
            elif user_input == "/clear":
                client.clear_history()
                print("История диалога очищена.")
                continue
            elif user_input.startswith("/model "):
                new_model = user_input.split(" ", 1)[1]
                client.model = new_model
                print(f"Модель изменена на: {new_model}")
                continue

            # Генерация ответа
            print("Ассистент: ", end="", flush=True)
            reply = client.generate(user_input, system_prompt=system_prompt)
            print(reply)

        except KeyboardInterrupt:
            print("\nЗавершение работы.")
            break

if __name__ == "__main__":
    main()
```

## Особенности реализации
- **Локальная работа** – не требуется API-ключ, всё выполняется на вашем компьютере.
- **Поддержка диалога** – модель помнит предыдущие сообщения (история хранится в `OllamaClient`).
- **Смена модели** – можно легко переключиться на другую модель без перезапуска.
- **Обработка ошибок** – если Ollama не запущен, выводится понятное сообщение.

## Преимущества
- Работает офлайн (после загрузки модели)
- Бесплатно, без ограничений по количеству запросов
- Полный контроль над данными (ничего не уходит в облако)
- Лёгкое расширение (добавить токенизацию, сохранение истории и т.д.)

## Ограничения
- Требуется установка Ollama и модели (около 2–4 ГБ дискового пространства)
- Скорость генерации зависит от мощности вашего CPU/GPU
- Качество ответа определяется выбранной моделью

## Возможные улучшения
- Добавить сохранение/загрузку истории в файл
- Реализовать режим однострочного запроса (без диалога)
- Поддержка потоковой передачи (streaming) для посимвольного вывода
- Интеграция с системными командами (например, /run python ...)
