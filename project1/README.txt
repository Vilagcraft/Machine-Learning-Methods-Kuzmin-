Для обучения нейросети запустите команду:
bash
python app.py train --data_dir ./data --epochs 20 --model_path model.h5


--data_dir – путь к корневому каталогу с подкаталогами классов.
--epochs – количество эпох (по умолчанию 20).
--model_path – имя файла для сохранения модели (по умолчанию model.h5).

В процессе обучения:
Данные автоматически разделятся на обучающую (80%) и валидационную (20%) выборки.
Будет применена аугментация для улучшения обобщения.
Лучшая модель (по точности на валидации) сохранится в указанный файл.
После обучения будут показаны графики потерь и точности, а также итоговая точность на валидации.
В текущем каталоге появится файл class_names.txt со списком классов (порядок соответствует предсказаниям).

Для распознавание нового изображения
поместите его в папку /new_image
bash
python app.py predict --image_path ./new_image.jpg --model_path model.h5 --class_names class_names.txt

--image_path – путь к изображению.
--model_path – путь к обученной модели (если не указан, используется model.h5).
--class_names – файл с именами классов (создаётся при обучении).

Программа:
Загрузит изображение, приведёт к размеру 128×128 и нормализует.
Выполнит предсказание.
Покажет изображение с подписью предсказанного класса и уверенностью.
Выведет вероятности для всех классов.

Примечания:
Модель представляет собой полносвязную нейронную сеть (feed-forward) с тремя скрытыми слоями (512, 256, выходной с softmax). Это многослойный персептрон.
Для работы с цветными изображениями они преобразуются в вектор размером 128×128×3 = 49152, что может быть требовательно к памяти. При необходимости уменьшите IMG_SIZE.
Аугментация помогает избежать переобучения при малом количестве данных.
Для оценки качества используется точность (accuracy) на отложенной валидационной выборке.




Важные замечания
Если фото не принадлежит ни одному из классов, модель всё равно выдаст какой-то класс с низкой вероятностью (например, 0.35). Ориентируйтесь на значение уверенности — если оно ниже 0.6–0.7, результат ненадёжен.
Убедитесь, что файлы model.h5 и class_names.txt находятся в папке проекта (они создались при обучении).
Если вы использовали формат .keras при сохранении, укажите --model_path model.keras соответственно.
Теперь вы можете тестировать модель на любых фото кошек, собак, птиц или рыб.
