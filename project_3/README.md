# Рекуррентная нейронная сеть для восстановления изображений

Простая реализация ассоциативной памяти на основе рекуррентной нейронной сети (SimpleRNN).  
Модель обучается на 10 эталонных бинарных изображениях размером 10×10 пикселей (крест, рамка, полосы, шахматная доска, диагональ, круг, буква T, уголок, зигзаг) и способна восстанавливать зашумлённую версию как эталонного, так и любого пользовательского изображения.

## Возможности

- Генерация 10 встроенных эталонных образов.
- Автоматическое создание обучающей выборки (150 зашумлённых копий на образ, уровень шума 25%).
- Быстрое обучение (1–2 минуты на обычном CPU).
- Восстановление зашумлённого изображения после обучения.
- Загрузка пользовательского изображения (любой формат, автоматически приводится к 10×10) и его восстановление.

- **Python 3.8 – 3.11**

  - `tensorflow` (2.x)
  - `numpy`
  - `matplotlib`

# Ответы на вопросы

### 1. Ассоциативная память, её виды

**Ассоциативная память** — это способность нейронной сети восстанавливать полный образ (или наиболее близкий эталон) по частичному, искажённому или зашумлённому входному сигналу. В отличие от адресной памяти (где данные извлекаются по адресу), ассоциативная память обращается по содержанию.

**Виды ассоциативной памяти:**
- **Автоассоциативная** — восстанавливает сам образ, поданный на вход (например, «очищает» зашумлённую картинку). Реализуется сетями Хопфилда, автоэнкодерами.
- **Гетероассоциативная** — отображает входной образ в другой образ (например, по имени человека выдает его фотографию). Реализуется сетями Хэмминга, двунаправленными ассоциативными сетями (BAM).

### 2. Восстановление образов при помощи модели «персептрон с узким горлом»

«Персептрон с узким горлом» (bottleneck perceptron) — это частный случай **автоэнкодера** (autoencoder). Архитектура: входной слой (например, 100 нейронов) → скрытый слой с малым числом нейронов (узкое горло) → выходной слой (той же размерности, что вход). Сеть обучается восстанавливать входной сигнал на выходе, проходя через сжатое представление.

**Принцип восстановления образов:**
- На этапе обучения сеть вынуждена находить компактные признаки (кодирование), достаточные для реконструкции исходного изображения.
- Зашумлённый образ подаётся на вход; проходя через узкое горло, шум частично отфильтровывается, а на выходе сеть генерирует очищенный образ (ближайший к обучающим данным).
- Такой подход эффективен для шумоподавления и восстановления пропущенных частей изображения.

### 3. Рекуррентная нейронная сеть Хопфилда. Принципы обучения и функционирования

**Сеть Хопфилда** — полносвязная рекуррентная сеть с симметричными весами без самосвязей. Используется как автоассоциативная память.

#### Обучение (запоминание образов)
Правило Хебба: каждый образ (бинарный вектор) вносит свой вклад в матрицу весов.  
Для запоминания M образов ξ(1), ξ(2), ..., ξ(M) веса вычисляются по формуле:

w_{ij} = \frac{1}{N} \sum_{m=1}^{M} \xi_i^{(m)} \xi_j^{(m)}, \quad i \ne j, \quad w_{ii}=0

где N — размерность вектора. Веса симметричны (w_ij=w_ji).

#### Функционирование (восстановление)
- На вход подаётся искажённый или неполный вектор.
- Состояния нейронов асинхронно обновляются по правилу:  
  s_i(t+1) = \text{sign}\left( \sum_{j} w_{ij} s_j(t) - \theta_i \right)
- Сеть сходится к локальному минимуму энергии (одному из запомненных образов или ложному аттрактору).
- **Ограничение:** ёмкость сети ≈ 0.15N (например, для 100 нейронов — не более 15 образов).

### 4. Рекуррентная нейронная сеть Хемминга. Принципы обучения и функционирования

**Сеть Хэмминга** предназначена для минимизации расстояния Хэмминга между входным вектором и запомненными эталонами. Состоит из двух слоёв.

#### Архитектура
- **Первый слой** (слой сравнения): вычисляет скалярное произведение входного вектора с каждым эталоном (фактически — сходство). Количество нейронов равно числу эталонов M.
- **Второй слой** (слой максимизации, конкурентный): реализует латеральное торможение («winner-take-all»). Нейроны соревнуются, и в итоге остаётся активным только тот, который соответствует эталону, ближайшему к входу (по Хэммингу).

#### Обучение
Веса первого слоя фиксируются равными эталонным образам (прямое запоминание без итеративного обучения). Веса второго слоя задаются так, чтобы обеспечить конкурентное подавление (например, сильные ингибиторные связи между нейронами).

#### Функционирование
- На вход подаётся зашумлённый образ.
- Первый слой вычисляет меру близости к каждому эталону.
- Второй слой за несколько итераций выделяет единственный наиболее подходящий эталон.
- **Преимущество:** меньше ложных аттракторов, чем у сети Хопфилда, работает быстрее.

### 5. Учёт контекста в рекуррентных моделях

**Контекст** в рекуррентных сетях — это информация о предыдущих состояниях, которая влияет на обработку текущего входа. Простые RNN имеют «короткую память» из-за затухающих градиентов. Для учёта долгосрочного контекста используются специализированные архитектуры:

- **LSTM (Long Short-Term Memory)** — содержит ячейки памяти и три управляющих вентиля (забывания, входа, выхода). Позволяет сохранять информацию на сотни шагов.
- **GRU (Gated Recurrent Unit)** — упрощённый вариант LSTM с двумя вентилями.
- **Двунаправленные RNN (BiRNN)** — учитывают как предшествующий, так и последующий контекст (например, для обработки всего предложения целиком).
- **Внимание (Attention)** — механизм, который позволяет модели динамически выбирать релевантные части входной последовательности при формировании выхода.

Учёт контекста критичен для задач машинного перевода, анализа тональности, генерации текста и восстановления последовательностей.

### 6. Практическое применение рекуррентных моделей

Рекуррентные нейронные сети (включая LSTM и GRU) широко применяются там, где данные имеют последовательную структуру:

| Область | Примеры задач |
|---------|----------------|
| **Обработка естественного языка** | Машинный перевод, генерация текста, анализ тональности, вопросно-ответные системы |
| **Речь и аудио** | Распознавание речи, синтез речи, музыкальная генерация |
| **Временные ряды** | Прогнозирование курсов акций, погоды, спроса на электроэнергию |
| **Медицина** | Анализ ЭКГ/ЭЭГ, мониторинг состояния пациента по временным данным |
| **Компьютерное зрение** | Распознавание действий в видео, captioning (описание видео), трекинг объектов |
| **Ассоциативная память и восстановление образов** | Очистка зашумлённых изображений (как в данной работе), восстановление пропущенных пикселей, исправление ошибок в передаче данных |
| **Рекомендательные системы** | Моделирование последовательности действий пользователя для предсказания следующего клика/покупки |

**В данной программе** мы используем простую RNN не для обработки временных рядов, а как ассоциативную память: входная последовательность (растр 10×10, развёрнутый в 100 шагов) проходит через рекуррентные слои, и сеть учится восстанавливать исходный эталонный образ из его зашумлённой версии.
