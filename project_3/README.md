```markdown
# Рекуррентная нейронная сеть для восстановления изображений

Простая реализация ассоциативной памяти на основе рекуррентной нейронной сети (SimpleRNN).  
Модель обучается на 10 эталонных бинарных изображениях размером 10×10 пикселей (крест, рамка, полосы, шахматная доска, диагональ, круг, буква T, уголок, зигзаг) и способна восстанавливать зашумлённую версию как эталонного, так и любого пользовательского изображения.

[figure_2](https://github.com/Vilagcraft/Machine-Learning-Methods-Kuzmin-/blob/main/project_3/demo/Figure_2.png)

## Возможности

- Генерация 10 встроенных эталонных образов.
- Автоматическое создание обучающей выборки (150 зашумлённых копий на образ, уровень шума 25%).
- Быстрое обучение (1–2 минуты на обычном CPU).
- Восстановление зашумлённого изображения после обучения.
- Загрузка пользовательского изображения (любой формат, автоматически приводится к 10×10) и его восстановление.

## Требования

- **Python 3.8 – 3.11**
- Установленные библиотеки:
  - `tensorflow` (2.x)
  - `numpy`
  - `matplotlib`
  - `pillow`

## Установка и запуск на Windows

1. **Установите Python**  
   Скачайте с [официального сайта](https://www.python.org/downloads/). При установке обязательно отметьте опцию **«Add Python to PATH»**.

2. **Откройте командную строку** (Win + R → `cmd` → Enter).

3. **Установите необходимые пакеты** (одна команда):  
   ```cmd
   pip install tensorflow numpy matplotlib pillow
   ```

4. **Скачайте файл программы**  
   Сохраните код из файла `rnn_recover.py` (или скопируйте текст из репозитория) в любую папку, например `C:\rnn_project\`.

5. **Запустите обучение и восстановление**  
   ```cmd
   cd C:\rnn_project
   python rnn_recover.py
   ```

6. **Что произойдёт после запуска**  
   - Программа сама сгенерирует 10 эталонов и обучит сеть (20 эпох).  
   - По окончании обучения появится окно с тремя картинками: оригинал (крест), его зашумлённая копия и восстановленный результат.  
   - Затем программа спросит: *«Хотите восстановить своё изображение? (y/n)»*.  
   - Введите `y` и укажите полный путь к любому файлу (PNG, JPG, BMP).  
   - Изображение будет приведено к 10×10, зашумлено (30% инвертированных пикселей) и восстановлено сетью.  
   - Результат отобразится в новом окне.

## Пример работы (рисунок `figure_2.png`)

На рисунке ниже показан типичный результат:  
- **Слева** – оригинальное пользовательское изображение (буква «А» 10×10).  
- **По центру** – то же изображение с добавленным шумом (30% пикселей изменены).  
- **Справа** – то, что восстановила нейронная сеть (ближайший запомненный эталон – крест или рамка).  

> *Обратите внимание:* сеть восстанавливает только те образы, на которых обучалась. Для улучшения результата замените встроенные эталоны на свои (поместив минимум 5 изображений в папку `user_imgs` рядом со скриптом).

## Примечания

- Модель простая и быстрая – не предназначена для высокоточного распознавания сложных объектов.
- Для обучения на своих изображениях создайте папку `user_imgs` в той же директории, что и скрипт, и поместите туда не менее 5 чёрно-белых (или цветных) картинок. Они автоматически будут использованы вместо встроенных эталонов.
- Если вы хотите получить осмысленное восстановление конкретного символа (например, буквы «А»), обязательно включите этот символ в обучающий набор.

### Как добавить фото `figure_2.png` в репозиторий
